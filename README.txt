📊 АНАЛИЗАТОР ДАННЫХ с графическим интерфейсом

🚀 БЫСТРЫЙ СТАРТ:

1. Установите зависимости:
   python install.py
   ИЛИ
   pip install -r requirements_gui.txt

2. Положите ваши файлы в папку 'data/'

3. Запустите программу:
   Windows: run_analyzer.bat
   Linux/Mac: ./run_analyzer.sh
   Или: python main.py

📁 ПОДДЕРЖИВАЕМЫЕ ФОРМАТЫ:
   • CSV (.csv)
   • Excel (.xlsx, .xls)
   • Parquet (.parquet)
   • JSON (.json)
   • Текстовые файлы (.txt)

✨ ВОЗМОЖНОСТИ:
   • Автоматический анализ всех файлов в папке
   • Обнаружение и обработка ошибок
   • Создание отчетов в HTML
   • Визуализация данных
   • Очистка и сохранение данных

💡 СОВЕТЫ:
   • Для больших файлов (>100 МБ) может потребоваться время
   • Excel файлы должны быть без сложного форматирования
   • Русские кодировки поддерживаются автоматически
